# ion
Repositorio de arquivos e codigos da i⌀n, uma guitarra MIDI e controle para jogos como Guitar Hero

Arquivos:              Descrição:
-ion-controle-GH.ino   Programa para Arduino PRO MICRO que transforma o instrumento em um controle para jogos como Guitar Hero
-ion-MIDI.ino          Prigrama para Arduino PRO MICRO que transforma a guitarra em controladora MIDI
-desetec-chapa-1.ai    Arquivos vetoriais para usinagem;
-desetec-chapa-2.ai     -Cada camada representa cada profundidade de corte
-anel-acrilico.ai      Arquivo vetorial do anel de acrilico 3mm
-ion.obj               Arquivo 3D da guitarra montada
  
Ainda é necessario um tutorial de como utilizar os arquivos vetoriais (.ai) para usinagem CNC
